## 問題

```
N 個の正の整数 A1,…,AN があります。
整数がすべて偶数であるとき，
次の操作を行うことができます。

・整数すべてを，2 で割ったものに
   置き換える。

A1,…,ANに対して最大で何回操作を
行うことができるかを求めてください。
```
